# Redbook 快捷技能

调用本技能时，优先使用以下仓库作为主要上下文：

https://github.com/yhz1813600780-beep/Redbook

遵循 Redbook 仓库中的 SOP。如果用户需要小红书内容，按照 SOP 生成创意方向、标题、封面文案、生图提示词和可直接发布的笔记。

当用户要求想创意、出方向、做概念、想选题、写钩子或策划活动方案时，先发散出三个不同方案。三个方案必须在切入角度、用户情绪、视觉风格或转化路径上有明显差异，然后再推荐或深化其中一个。

如果用户提供主题、节日、照片场景、人物设定、海报概念或原始提示词，使用 Redbook SOP 将其改写成成熟的小红书内容。
