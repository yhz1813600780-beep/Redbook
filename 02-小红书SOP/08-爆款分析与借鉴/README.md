# 爆款分析与借鉴

这里用于长期沉淀小红书爆款分析方法，不只是看“哪条火了”，而是要拆出可复用的因子。

## 重点方向
- 借鉴爆款选题
- 拆解爆款因子
- 整理钩子和开头
- 观察标题、封面、结构、互动设计
- 把分析结果反哺到内容生产 SOP

## 建议使用方式
1. 先用 `SOP-007_分析_爆款分析与借鉴流程.md` 明确分析步骤
2. 每拆一篇爆款，按 `05-模板库/爆款分析模板.md` 填一次
3. 定期把重复出现的模式沉淀成“爆款因子”
4. 如果爆款因子已经能稳定复用，再反哺到选题、标题封面、内容生产或复盘 SOP

## 素材反哺路径

新增爆款素材时，建议按下面顺序处理：

1. 先查本目录是否已有同类案例或分类归纳。
2. 没有重复案例时，再新增 `案例-编号_主题_爆款拆解.md`。
3. 有重复案例时，优先补充已有案例或分类文件，不要重复建档。
4. 拆解完成后，至少沉淀 1 个可迁移因子和 3 个可测试选题。
5. 如果新规律影响现有执行方法，同步更新对应 SOP，并在 `CHANGELOG.md` 记录。
