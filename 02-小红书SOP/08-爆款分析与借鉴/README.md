# 爆款分析与借鉴

这里用于长期沉淀小红书爆款分析方法，不只是看“哪条火了”，而是要拆出可复用的因子。

## 重点方向
- 借鉴爆款选题
- 拆解爆款因子
- 整理钩子和开头
- 观察标题、封面、结构、互动设计
- 把分析结果反哺到内容生产 SOP

## 建议使用方式
1. 先用 `SOP-007_分析_爆款分析与借鉴流程.md` 明确分析步骤
2. 每拆一篇爆款，按 `05-模板库/爆款分析模板.md` 填一次
3. 定期把重复出现的模式沉淀成“爆款因子”
