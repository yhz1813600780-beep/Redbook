### 原 prompt 结构
- 输入：人物肖像、毕业照、校园主题。
- 主体结构：巨大的侧脸剪影作为外轮廓。
- 内部世界：校园、标志性建筑、角色关系、象征符号、道具、氛围。
- 视觉风格：剪影轮廓填充式叙事合成、双重曝光联想、电影海报、梦幻水彩插画。
- 质感要求：柔和空气透视、轻雾、纸张颗粒、边缘飞白、刷痕、留白、高级克制。
- 负面约束：不要杂乱、不要硬拼贴、不要模板化背景、不要廉价奇幻素材。

## 4. 点击前拆解
- 标题承诺：5 秒生成毕业神图，速度快、效果强、还附教程。
- 封面结果：成品不是普通 AI 图，而是纪念品级海报，看起来像电影海报或校园收藏海报。
- 点击理由：毕业季用户有明确需求，既想要高级纪念照，又缺少设计和修图能力。
- 收藏理由：正文直接给出 prompt，用户会保存下来等自己有照片时复用。

## 5. 钩子与开头拆解
- 开头钩子：把毕业照的拍的图片选四个，丢给工具并喂 prompt，就能火速生成大片。
- 开头类型：场景痛点 + 低门槛动作 + 高价值结果。
- 留人原因：用户马上知道自己需要准备什么、能得到什么、为什么比传统拼图更高级。
- 可借鉴句式：
  - 选【数量】张【场景照片】，丢给 AI，直接生成【高价值成品】。
  - 别再做普通拼图了，用【人物轮廓 + 场景世界观】做一张收藏级海报。
  - 【人生节点】照片别只发九宫格，试试这个【电影海报级】玩法。
  - 只需要【素材要求】，剩下交给 prompt。

## 6. 正文结构拆解
- 内容顺序：结果吸引 -> 输入要求 -> 工具动作 -> 成品价值 -> 完整 prompt -> 标签分发。
- 关键转折点：“传统拼图排面不够大”，把用户从“我想晒照片”带到“我想做一张更有纪念感的作品”。
- 最容易收藏的部分：完整 prompt、素材要求、负面约束。
- 最容易引发评论的部分：工具是什么、怎么上传照片、能不能换学校/专业/城市、中文能不能出字、是否支持情侣/多人照。

## 7. 爆款因子提炼
- 选题因子：毕业季强时间窗口 + AI 工具新鲜感 + 个人纪念照刚需。
- 标题因子：时间极短“5s” + 工具名“Image2” + 高结果词“神图” + 附教程降低门槛。
- 封面因子：先展示最终海报，不展示操作界面；让用户被结果吸引。
- 钩子因子：4 张照片即可生成，比“学设计”轻很多。
- Prompt 因子：结构清楚，包含主体、场景、风格、质感、负面约束，容易迁移到其他人生节点。
- 情绪因子：毕业是人生节点，用户愿意为纪念感、仪式感、独特性买单。
- 互动因子：评论区天然会围绕学校、专业、工具、prompt 改写、照片数量追问。
- 转化因子：可延伸到毕业照 prompt 包、AI 海报模板、学校定制、个人 IP 海报、节日纪念照。

## 8. 可复用创作公式
### 公式 1：人生节点海报型

用【人物轮廓】装进【地点/经历/象征物】，  
生成一张【收藏级纪念海报】。

### 公式 2：少量素材高结果型
只要【数量】张照片，  
AI 自动生成【高价值成品】，  
适合【具体场景】。

### 公式 3：Prompt 公开收藏型
标题给结果，正文给完整 prompt，  
让用户不仅想看，还想保存复用。

### 公式 4：传统方案替代型
别再做【传统低级方案】了，  
用【AI 新方案】做出【更有排面/更高级/更有纪念感】的结果。
